Images and Multimedia

  • <img>: embed an image
  • <map>: define an image map (clickable link area). Use with <area>
  • <area>: defines clickable areas within an image map. e.g. separate sections of same image can have different hyperlinks
  • <audio>: embed sound content. use with src attribute. can be destination for streamed media using MediaStream
  • <video>: embeds a media player which supports video playback
  • <track>: used as child of media elements. for timed text tracks (or time-based data) e.g. subtitles or lyrics
