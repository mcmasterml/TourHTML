CSS Notes

px, em, rem, vw, vh, and %

px -- ensure consistent results

em -- relative to parent element's font size

rem -- relative to the root (HTML) font size

vw -- view width, always based on view (changing zoom changes size)

vh -- view height, always based on view (changing zoom changes size)

% -- percentage of parent element's width or height
