Block, Inline, and Inline-Block Display

Newline or Sameline?

a <p> element has a block layout

  • next html element will begin on a New Line
  • takes up entire width of parent container
  • you can set height, width, margin, padding, and border

a <span>, <img> element has an inline layout

  • same line, nests with other elements
  • you cannot adjust height, width, margin, padding, and border

a <p> element has a block layout

  • next html element will begin on a New Line

CSS Notes

px, em, rem, vw, vh, and %

px -- ensure consistent results

em -- relative to parent element's font size

rem -- relative to the root (HTML) font size

vw -- view width, always based on view (changing zoom changes size)

vh -- view height, always based on view (changing zoom changes size)

% -- percentage of parent element's width or height
