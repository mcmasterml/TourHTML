HTML Text Content

Identify the purpose of content, important for accessibility and SEO

  • <blockquote>: extended quotation, usually accompanied by indentation. use with <cite> for URL source attribution
  • <div>: generic container. Has no effect on content or layout until styled in some way, (e.g. CSS or layout model like flexbox applied to its parent element)
  • <dl>: description list. container for terms <dt> and descriptions <dd>. examples: a glossary or dictionary (key-value pairs)
  • <dt>: description term. use in description lists <dl>
  • <dd>: description definition. use in description lists <dl> after a term<dt>
  • <figure>: represents self-contained content. images, captions, code blocks, poems, quotes. use with <figcaption>
  • <figcaption>: caption or legend describing content in its parent <figure> element.
  • <hr>: purely semantic "horizontal rule" or "line". thematic break. have to draw a horizontal line in CSS, as this is now purely semantic.
  • <ol>: ordered list
  • <ul>: unordered list
  • <li>: list item
  • <menu>: semantic alternative to <ul>, no real difference
  • <p>: paragraph
  • <pre>: rofl copter. preformatted monospaced text, represented exactly as written in the HTML file (e.g. rofl copter)
